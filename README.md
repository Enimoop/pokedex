# Pokedex App

Application web de gestion de Pokémons avec interface React côté frontend et API Symfony côté backend.

---

## Fonctionnalités

- Affichage paginé des Pokémons (48 par page)
- Ajout, modification, suppression de Pokémons
- Gestion des types et sexes associés
- Interface responsive avec adaptation du layout selon la taille d’écran
- Popup de détails/modification des Pokémons
- Mode d’interaction : sélection, ajout, modification, suppression
- Validation et gestion des erreurs côté client et serveur

---

## Technologies utilisées

- Frontend : React (Hooks, composants fonctionnels)
- Backend : Symfony 6 + API Platform / Doctrine ORM
- Base de données : MariaDB/MySQL
- Communication : REST API JSON via fetch
